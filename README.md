Welcome to My Budget Buddy!
This application is designed to help students take control of their finances. MBB is not like any other financing application,
this is an application where they use more than standard expense tracking, using 2 advanced features to maximise the users engagement and discipline.

Feature 1 : 
The Envelope Budgeting System ✉️
- This is inspired by the traditional budgeting system of physical cash.
- The envelope budgeting system allows users to add certain amounts of money to be allocated to specific categories, whichs helps limit overspending.

The envelope system uses unique colours, helping users be able to select the  colours and ensure that they know what colours are linked to which
envelope. It also uses a Total Spending Matrix, which shows the total spending of a user over a set time period, and displays the total on the tracking bar.
This system has different levels as to what a user can do:
- Free Mode: a user is limited to 3 custom envelopes. If a user tries to add a 4th envelope, the system has a prompt to let the user know that they
  need to upgrade their subscription.
- Premium Mode: a user can have unlimited envelopes, and has access to premium features of the application.

Feature 2: 
The Achievements 🏆
- This is gamification of the application, helping with financial responsibilities, while rewarding users with badges as they build healthy habits.
- When certain milestones are hit, the application updates the cloud and shows a badge in the achievements page.

The achievements are on a page, specifically showing what was achieved and what they could get in the future. When a user creates their first envelope,
the application will update the database with the achievement, and will show the badge on the Achievements page. When a user logs their first expense, 
the application will update the database with the achievement, and will show the badge on the achievements page.
The achievements page will show grayed out badges on the Achievements Page, and when an achievement is secured, it will change colour, to full colour.
